# spotify-clone# 🎵 Spotify Web Player Clone

A fully responsive frontend clone of the Spotify Web Player user interface. This project replicates the iconic dark-mode layout of Spotify, featuring the side navigation panel, dynamic content music cards, and a fully stylized custom playback control bar.

## 🚀 Features

* **Spotify Layout Architecture:** Built using CSS Flexbox and Grid to achieve the classic three-column layout (Sidebar, Main Content, Media Player).
* **Custom Media Controls:** Includes a highly customized progress and volume bar matching Spotify’s design token system.
* **Interactive UI:** Smooth hover states, custom transitions, and dynamic opacity fading implemented across all primary interaction links and icon sets.
* **Responsive Architecture:** Media queries implemented to seamlessly handle varying viewport widths down to mobile breakpoints.

## 🛠️ Tech Stack

* **HTML5:** Semantic markup structure.
* **CSS3:** Flexbox, Grid, custom Webkit slider configurations, and UI positioning.
* **Font Awesome:** Scalable vector icon integration for consistent media control styling.
* **Google Fonts:** Montserrat typeface integration for accurate typography matches.
